Descrizione:

ID: Identificativo univoco della manutenzione

CODE: Tag univoco per legare la manutenzione alla pagina corretta

NAME, DESCRIPTION: Descrizione della manutenzione

TECHNICAL LEVEL: Livello del tecnico che esegue la manutenzione

CRITICAL LEVEL: Gravità della manutenzione: 0 = Solo informativa, 1 = Avvertimento significativo, 2 = Avvertimento bloccante

TYPE: Tipo di conteggio per sollevare la manutenzione: 0 = A tempo, 1 = A tempo di un registro macchina, 2 = A misura di un registro macchina

WARNING VALUE, ERROR VALUE: Valore limite per sollevare gli eventi di avvertimento e errore

NODE VALUE: Valore del nodo o registro della macchina per conteggiare la manutenzione

UM: Unità di misura del valore letto dal nodo per quanto riguarda il tempo, secondi, ore, giorni, etc

ACTIONS: 0=none,1=warning,2=maxvalue,3=both: ShowMessage, SendMail, BlockMachine
