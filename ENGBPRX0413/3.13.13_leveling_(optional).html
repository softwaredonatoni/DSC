3.13.13 LEVELING (OPTIONAL)

    This feature allows the operator to execute a planing on the piece.

    Parameters

    Pass Extension

    Each cut generated from this working is prolonged, both at the head and at the tail, of the length indicated by this parameter.

    Execution of working

    Starting from slab thickness set in material parameters, to reach the objective thickness set in the parameters of the working, the piece is worked in the following way:

    1. As first, parallel cuts are executed from left to right at a distance set in the parameters and executed with the parameters of the linear cut.

    2. Subsequently, cuts along the Y axis to create a chessboard always executed with the parameters of the linear cut.

    Warning

    WARNING!The initial slab thickness must be set as material thickness (it is possible then to add the keyboard for a greater precision).The target thickness is set in the processing parameters (and refers to the bench quote).

    Info

    NOTEThe following working can also be executed on commercial pieces.

    Slab processing parameters

    The operator can change the parameters related to the flattenings from the panel of the machining parameters of the slab.

    Flattening Parameters cuts

    Description parameters

    Distance between horizontal passes
    Represents the distance between two parallel passes along the horizontal direction.
    The button allows to decide if the passes must be executed from top to bottom or vice versa.

    Distance between vertical passes
    Represents the distance between two parallel passes along the vertical direction.
    The button allows to decide if the passings must be executed from left to right or vice versa.

    Flattening quote
    Set the thickness that the material must have after the leveling.

    Rotate tool 90 degrees
    If enabled, the leveling processing is executed with A axes to 90 degree.
