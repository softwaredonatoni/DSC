3.13 MODIFY PIECE

    This panel allows the operator to apply different types of working to a designated piece.
    Upon opening, the following screen is visible:

    Warning

    WARNING!Some of the functions that will be described in this section are also present on the main page of Parametrix, but with a difference.If a piece, or a cut, is modified through the 'Modify Piece' panel, all pieces of the selected row will be modified accordingly with the same workings.Conversely, if the working is set from the work area, only the selected piece will be modified.

    Working types

    Through the 'Edit Piece' panel it is possible to apply one or more of the following workings, which will be individually illustrated later. In particular, Parametrix allows to:

    Miter Cut
    Modify the cut inclination, both under square and above square.

    Drip system
    Add a drip system to the side of a cut.

    Add
    Add a cut parallel to a side of the piece.

    Delete
    Delete the cuts of the piece.

    Offset
    Change the dimensions of the piece applying offset to its sides.

    Hole
    Add holes in the material, even to remove what the disk was not able to cut.

    Milling
    Replace workings with disk with milling cuts.

    Rim
    Apply rims with milling. It is used, for example, to create recesses for kitchen top sinks.

    Lower Cut - (Optional)
    Perform a cut on the lower surface of the piece using a special mechanical optional.

    Downstand
    Add risers or rims for the sides of piece.

    Curbs
    Add a chamfered cut to a piece's side.

    Surface - (Optional)
    Apply superficial passes with vertical cuts and horizontal disk.

    Leveling - (Optional)
    Apply a leveling cycle with vertical cuts and horizontal disk.

    Channel - (Optional)
    Add a draining channel on the piece using the disk.

    Fillet
    Round a corner or a side of the piece by applying a radius.

    Crop
    Cut the piece using as a reference a side, or a custom line.

    Internal break
    Add a internal notch to the piece, of rectangular, circular or elliptical shape.

    Drain board flutes
    Create grooves through milling.

    MaxBound - (Optional)
    Obtain the piece's maximum footprint.

    Mirror
    Obtain the specular version of the piece.

    Info

    NOTEDepending on the combination of optional, some of these buttons may not be present.

    List of cuts

    The piece cuts will be visible in the bottom left section of this panel.

    Through apposite subdivision, it is indicated if a cut belongs to the external perimeter or to one of the internal perimeters. Thanks to this classification, it is possible to show/hide the list of the cuts belonging to a polygon by pressing on the icon of the corresponding section.

    As you can see from the image, in each line are shown some information about the cut that represent. In particular:

    Manage working
    Button with which it is possible to interact with the cut and apply the current working.
    In the main page it allows to activate/deactivate the cut.

    Cut information

    • T - indicates the type of cut ('Straight', 'Curve', 'Milling', etc…).

    • D - indicates the cut length.

    • I - indicates the inclination of the cut.

    • P - indicates the cut's penetration.

    Processing information

    • Working type ('Linear', 'Arc', 'Inclined', etc…).

    • Steps mode (Full Step or Step mode).

    • Speed of cutting advance.

    • Diameter of the tool used for the machining (Disk, Milling cutter, etc…).

    Info

    NOTEThese information change based on the typology of the cut that describe. Not always are showed all information.For example, for holes it is possible viewing exclusively the penetration in the material.

    Info

    NOTEIf on a part of the piece no cut is present, it is indicated with the wording 'Nothing'.

    Preview

    The main part of the 'Edit piece' functionality makes available to the operator a complete preview of the selected piece, showing all workings currently applied.

    It is possible to interact with the displayed piece simply by moving the preview, or using the buttons to modify the zoom that are found on its right.

    Furthermore, in this section are present other functions that we will describe below.

    Mark
    Is provided the possibility to change the mark of the selected piece.

    Pieces Number
    Is provided the possibility to modify the number of pieces executed and the total number of pieces.

    Add favorite
    The piece viewed in the preview will be inserted in the list of preferred pieces, together with all the changes applied at the time of insertion in the preferred.

    Selection of cuts

    It is possible to select a cut directly from the preview, observing the change of its color to yellow to confirm the selection, or select the corresponding line inside the cuts list available in the menu in the lower left.

    Clicking on the green dot present on the left of each element of the list, it will be possible to disable the relative cut.
    In this case the badge will become red and the preview of the piece will be updated with the color of the disabled cut that will change into grey.

    Lower bar

    In this bar are put at disposal some functionalities that simplify the enjoyment of the Workings from Modify Piece

    Cuts
    Visible the list of piece's cuts.
    It is the default view of this panel.

    Sides
    This button allows to visualize the sides lenght of the Piece inside the list, instead of the cut lenght. The selection directly on the image or on the list, will color of yellow the side of the selected piece.

    Info

    NOTEIn this mode it is not possible add workings to apply on the piece.

    Chronology
    Visible the history of the operations applied to the piece.
    It also allows resetting the piece to the desired operation.

    Cancel
    Allows to undo the changes made on the piece.
    Multiple presses of this button allow to return to the initial conditions of the piece.

    Warning

    WARNING!The button works as long as the 'Edit Piece' panel remains open on the piece under examination. If the panel is closed you lose the possibility to undo the changes on the piece in question, even if the panel were opened again.

    Change Visible
    Enable or disable certain piece entities to facilitate the visibility and the modification of the piece.

    Chronology

    Allows to visualize all modifications applied to the current piece and, if necessary, allows to restore the current piece to a previous state.
    If changes have been applied to the current piece, these will be displayed in the list on the left:

    In the menu on the left are displayed the states of the piece working.

    State processing
    Identifies a precise moment of the piece processing. Allows to visualize:

    • Name of the current piece;

    • Date of work's application;

    • Name of applied processing;

    • Icon that identifies the working done.

    The current piece’s state without changes is named 'Default'.
    Selecting a voice from the left menu, the corresponding preview is shown at the moment of work's application.

    Press the confirm button, it will cause the piece to be restored to the selected state. In this way, the subsequent changes to that selected are removed.

    In the lower part of the panel are visible three utility buttons for piece viewing in the preview.

    Viewing Cuts
    Show the cuts on the figure.
    It is the active viewing by default.

    Display Measures
    Shows the measures of each side of the figure.

    Visible angle
    Show a corner of the figure with the respective measurements.

    Change Visible

    This mode makes available to the operator some buttons visible only when the 'visual change' is enabled.

    Each of the following buttons enables or disables one of the following piece's elements:

    Cuts
    Show/hide the viewing of cuts with disk on the preview.

    Holes
    Show/hide the viewing of the holes.

    Milling
    Show/hide the milling display.

    Rim
    Show/hide the viewing of Rims.

    Arrows on cuts
    Show/hide the viewing of the arrows that indicate the direction of the cuts with disk.

    Piece name
    Show/hide the piece name.

    Dynamic Tags
    Show/hide the piece's dynamic tags.
