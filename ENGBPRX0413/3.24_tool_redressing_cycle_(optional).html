3.24 TOOL REDRESSING CYCLE (OPTIONAL)

    Starting the tool redressing cycle, proceed to the redressing of the same.

    Button

    Tool sharpening cycle
    Allows to access to the Tool redressing panel.

    Redressing tool from parameters page

    Accessing the parameters section of Parametrix we will find the following window:

    By clicking on the icon highlighted in red in the previous image, a window will open similar to the one shown here below:

    From this window, it is possible to set the following parameters for the tool redressing.

    Parameters

    Increase Penetration
    Indicate the penetration value for each pass.

    Steps to step
    Indicate the number of steps.

    Penetration Speeds
    Determine the speed during the phase of the tool's penetration on the material.

    Forward Speed
    Indicate desired speed for cutting phase in incoming.

    Restart cycle
    Clicking on this button will restart the cycle.

    Width
    Width of material for the redressing.

    Height
    Height of the material for the re-edger.

    Thickness
    Thickness of the material for the reliving.

    Redress tool from manual control panel

    Accessing the panel of manual controls it is possible to go to perform the tool redressing.

    Clicking the icon highlighted in red in the figure below, a dialog box will be displayed that allows to start or cancel the procedure.
