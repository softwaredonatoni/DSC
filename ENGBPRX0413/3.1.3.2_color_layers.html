3.1.3.2 COLOR LAYERS

    Through this function is possible to color one or more cuts of pieces created from DXF.

    Buttons

    Change layer color
    Press on the button on each row to change the color associated to the corresponding layer.

    Procedure

    To associate a color to the cuts of a piece it is necessary:

    • Create a Layer on the CAD software;

    • Assign to him a color in the specific panel in Parametrix;

    • Draw a line (or curve) above the one that represents the cut that you wish to color inside the Layer.

    Once the piece is imported the cut above which the line was drawn on the Color Layer will be the color set in Parametrix (not of the one set in DXF).

    Info

    NOTEThe line drawn above the cut that you want to color must be completely superimposed on the line that represents the cut.
