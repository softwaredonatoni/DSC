3.21.7 OPERATIONS

    This function allows to apply orderings and operations customized to the cuts.

    Available Operations

    Sorting for processing
    It allows to order the cuts in the list based on the working type.

    Macro
    Allows to apply macros to the cuts.

    Single operations
    Allows to apply standard operations to the cuts.

    Workings order in head
    Allows to move to the top of the list all cuts associated to a working type.

    Sorting workings queue
    Allows to move to the bottom of the list all the cuts associated to a working type.

    Sort based on cuts
    Allows to change the order of the cuts according to the position of the cuts on the work area.

    Execution workings order

    Allows the operator to choose the execution order of the cuts based on the type of processing used.
    The list is composed by the workings requested to execute the cut.
    Changing the order of the workings changes also the order of the cuts of the list.

    Info

    NOTEThe working types shown within this list are related to those associated with the cuts present on the working area.Based on this, different entries can come.

    Lower bar

    It is possible to order the list items via some move buttons.

    Move
    Move the selected operation to the previous or next position.

    Go top/bottom
    Bring the selected operation to the top or bottom of the list.

    Macro

    It is possible to define and apply macros. Each one of them allows the operator to apply advanced sorting options on the cuts.
    The selection of a macro in the top part of the panel allows the operator to view all the operations contained in the same macro.

    Info

    NOTEThe macros available to the operator can be managed in the user parameters.According to this, different entries can come.

    Buttons

    Apply operation
    Allows to execute the selected operation.

    Single operations

    Some operations can be applied by pressing the green arrow that is seen on the right of each item in the list.

    Single operations available

    Move auto
    Apply the automatic handling function of the slab.

    All cuts out slab
    Allows the operator to extend all cuts with disk to the edge of the slab simultaneously.

    Order by tool
    Modify the order of cuts to group all those that can be executed with the same tool.

    Order by working type
    Modify the order of the cuts to group all those that require the same working.

    Shorten cuts with intersections
    Allows to shorten the cuts that intersect each other to avoid to ruin the inside of a piece.

    Single operations available (Optional)

    It is possible to set the presence of some single operations in relation to how the following optional are set.

    Unload piece

    With the Unload Piece function enabled, some single specific operations will be available.

    Unload pieces

    Unload all pieces with repetition

    Unload watermark with repetition

    Double disk

    With the Double disk feature enabled, some single specific operations will be made available.

    Extend cut for double disk

    Verify for double disk

    Workings sorting in head

    The operator can change the order of execution of the workings and put at first place one of the workings that he sees on this window.

    Info

    NOTEThe working types shown within this list refer to those associated with the cuts present on the working area.Based on this, different entries can come.

    Sorting workings in queue

    The operator can change the order of execution of the workings and put at the end one of the workings that is seen on this window.

    Info

    NOTEThe working types shown inside this list refer to those associated with the cuts present on the working area.Based on it, different entries can come.

    Sort based on cuts

    These operations allow to change the order of the cuts of the list based on some criteria that depend on their arrangement on the work area.

    Operations sorting based on cuts

    First bottom cuts
    Order the cuts performing first those that are found in bottom, and proceeding towards the top.

    First top cuts
    Order the cuts performing first those that are found in top, and proceeding towards the bottom.

    First right cuts
    Order the cuts performing first those that are on the right, and proceeding towards left.

    First left cuts
    Order the cuts performing first those that are on the left, and proceeding towards right.

    First horizontal cuts 0_
    Order the cuts performing first the horizontal cuts.

    First vertical cuts 90_
    Order the cuts performing first the horizontal cuts.

    First cuts with direction
    Sort the cuts giving precedence to those with greater translational rotation value.

    External internal
    Order the cuts performing first the most external and then always more towards the interior.

    Internal external
    Order the cuts performing first the most interior cuts and then always more towards the outside.

    Info

    NOTEFor these orderings, the starting point of the cuts is considered.
