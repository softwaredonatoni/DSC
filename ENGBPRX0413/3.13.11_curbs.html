3.13.11 CURBS

    This functionality allows to add a curbs of type bevel or of type beak (beak of owl) on the cuts of the selected piece.
    The interface for this function resembles the following image:

    Parameters

    To use this processing, it is necessary to first choose one of the two available modes, selecting the corresponding button:

    Info

    NOTEThe 'Top Corner' and 'Bottom Corner' modes are mutually exclusive: activation of one automatically excludes the other.

    Chamfer
    Selecting this function the cut will be executed on the upper corner of the piece's side.

    Lower Chamfer (Lower Chamfer)
    Selecting this function the cut will be executed on the bottom corner of the piece's side.

    The following parameters are common to both working types:

    X Quote

    Distance along X axes from piece's edge.

    Y Quote

    Distance along Y axis from the piece's edge.

    Inclination A

    Inclination angle of the cut expressed in sexagesimal degrees.

    Thickness

    The slab thickness is shown currently set in Parametrix to help the user apply the processing.

    Info

    NOTEThis parameter can not be changed directly from the curb section.

    Perimeter

    Activating this parameter the working will be executed on all the perimeter of the piece.

    Info

    NOTENext to the first three geometric parameters, there are square selection boxes, initially deactivated. It is necessary to activate one to specify which parameter to keep constant; in this way, providing a second value, the third is calculated automatically. The selection boxes cannot be enabled if the value of the corresponding parameter is equal to zero, in order to prevent the setting of geometrically inconsistent or invalid configurations.

    Execution of working

    Info

    NOTEIt is possible to apply the processing by pressing the button on the left of the suitable cut in the polygon list.A red icon with an 'X' indicates that the processing can not be applied to the desired cut.

    The cordolo working can be executed only on the row type cuts.

    For each ridge inserted successfully on the material, there is the update of the preview and the automatic insertion in the 'Additional works' section on the left.
