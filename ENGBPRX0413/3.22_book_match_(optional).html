3.22 BOOK MATCH (OPTIONAL)

    This functionality allows to have a rendering of the project, and to see in real time the background that will assume the pieces, so as to be able to position them on the slabs so as to make the designs created by the stone fit.

    Buttons

    Within the lower bar of the Parametrix main screen, there is the button to access the Book Match functionality.

    Book Match
    Allows to program the slabs in book match.

    Design mode

    There are two modes through which is possible program book match designs.

    Easy Match

    Designed for a sequential working.
    It allows to manage one slab at a time, and the overall view can be had only at the end of the working of all necessary slabs.
    Recommended for the programming on-board machine.

    Project Match

    Designed for the planning of large workings.
    Allows to manage more slabs simultaneously so as to plan all project with a overall view.
    Suggested for the programming in office.
