3.12 PIECES LIST

    When a or more pieces are created and imported on Parametrix, the program will adds them to the pieces list, located on the right section of the main screen.

    Through the pieces list, the operator can:

    • Insert the pieces on the working area.

    • Keep trace of pieces executed and of those still to work;

    • Apply some modifications to pieces.

    List items

    Each line of the list represents a piece that has been imported in Parametrix.

    As you can see in this image, in a line are contained the following information relative to the piece:

    Shape preview

    Simplified representation of the piece's geometry.

    Piece brand

    Label that is added to the piece to identify it.

    Info

    NOTEIn the case the operator does not choose the brand in creation phase, the program identifies the piece with the name of the file (DXF or CSV) from which it was imported, or with the measures of the sides if it is a rectangle.

    Number of pieces inserted and total

    Indicates the number of pieces of that line that have been inserted inside the work area, and the number total of pieces of that type that is possible to insert.

    Insert the pieces in the work area

    To insert the pieces in the working area you use the following button that is found in every row of the list:

    Insert pieces in the work area
    Allows to insert the selected piece inside the working area.

    The selected piece has been placed on the work area, and the number of pieces of this row entered has been updated.

    Info

    NOTEOnce reached the maximum number, the button 'Insert pieces' of that line disappear, because all the pieces of that line have been executed.

    Operations on the pieces of the list

    The buttons below the list allow to manage the list items. In particular:

    Clean piece list
    It allows to delete all contents of the pieces list.

    Info

    NOTECommercial pieces are not removed from the list.

    Warning

    WARNING!Will be removed from the work area all pieces connected to lines removed from the list.

    Hide/Show commercial pieces
    Allows to hide or view the lines of the list associated with commercial pieces.

    Warning

    WARNING!Hiding the lines associated with commercial jobs, all pieces linked to them present on the work area are removed.

    Delete piece
    Allows to select individually the pieces to delete from the list.

    Filters/Sort the pieces
    It is possible to apply a filter to the pieces of the list, or to order them according to different criteria.

    Additional piece properties
    The operator can assign and change the additional properties of the selected piece.

    Delete piece

    This function allows to delete the pieces from the list individually. After pressing the button, the program will ask the user to select the row to delete replacing the piece insertion arrow on the work area with a gum.

    Pressing this button, the corresponding line is eliminated, and then all the pieces that it represents.

    Warning

    WARNING!Deleting a line, all the pieces connected to it present in the working area are removed.
