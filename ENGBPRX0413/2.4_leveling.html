2.4 LEVELING

    The leveling cycle gives the possibility to perform a leveling of the bench or the material. This program is useful when the bench is made uneven by the cuts, or you want to equalize the thickness of the material.

    Leveling parameters

    Quote of the movement in relation to the X axis
    It is possible to set positive and negative values.

    Quote of the movement with respect to the Y axis
    It is possible to set positive and negative values.

    Measure displacement
    Indicate the distance between a cut and the next one in this field it's possible to set only positive value. To perform the leveling is advised to set a 'move value' about 80% compared thickness of the tool.

    Depth
    Indicate how much to descend in depth during the planing cycle with respect to the acquired quote.

    Z Step
    Indicates the maximum descent step into the material. Based on the leveling depth, the increment in Z can decrease so as to carry out all the passes at the same depth.

    Cutting speed in plunge
    Indicates the work speed on the Z axis during the automatic cycle, is expressed in (mm/Min).

    Speed of cutting forward
    Indicates the work speed on the XY plane during automatic cycle, it is expressed in (mm/Min).

    Modify tool origin
    Pressing this button is performed the switch between origin of disk calculated with line laser, or origin of drill calculated with cross laser.

    Execution of working

    The program 'SPIANATURA' does not need zeroing origin X, Y and Z.
    To determine the starting position, the position of the tool is considered at the moment when the cycle will be started.
    After setting the various parameters, move with the disk turned on to the maximum desired depth of the bench or material and start the cycle with the START CYCLE button.

    Info

    NOTEThe program can be used with the A axis inclination to ZERO degrees or to NINETY degrees to use tools like disks or milling cutters.
