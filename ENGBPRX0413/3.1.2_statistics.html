3.1.2 STATISTICS

    Through this window it is possible view the statistics of the work done by the machine.
    The statistics include also workings of single cut type, excavation, leveling and ISO execution.

    Statistics on video

    Through the button in the top left corner it is possible to access the video statistics window.

    Statistics on video
    Opens a window where you can view the statistics on video.

    It opens then a window that shows some information related to the workings executed in machine.

    Each entry in the list corresponds to the related data of a single working.
    At the bottom of the page is instead presented a summary of the important information of all workings visible.

    Info

    NOTEThe statistics are updated every time a new working is executed.

    Buttons

    Daily Statistics
    Show statistics of workings executed in current day.

    Monthly Statistics
    Show the statistics of the workings performed in the current month.

    Total Statistics
    Show statistics of all work done by the machine.

    Adjustable time interval
    Allows to set the time interval inside to view the statistics of the work carried out inside it.

    Columns to display
    Allows to select the columns to view.

    Save Statistics
    Allows to save the statistics currently visible on screen to a CSV file.

    Statistics for each machine (Optional)

    For the Parametrix version in office mode, it is possible to connect to more machines, and view the statistics of each of them.
    On the left section are visible all the machines connected to the device.

    Research

    This function allows to effect a research of the statistics inside of a time interval at choice.

    The operator can insert the work start date and work end date to delimit the time interval of interest.
    It is possible to establish the research precision using the buttons.

    Research for Day
    Allows filtering the statistics with daily precision.

    Search per month
    Allows to filter the statistics with monthly precision.

    Research by Year
    Allows to filter the statistics with annual precision.

    To export the statistics, use the button:

    Save Statistics
    Allows to export the statistics of the workings carried out in the established time interval to a CSV file.
    The file name will be composed of: current date _ hour, minutes, seconds when the file was created.

    Timer

    In this section is possible to manage the machine timers.

    Using the following buttons, the operator is able to manage manually a timer that can keep track of the time elapsed.

    Start Timer
    Allows to start the timer.

    Interrupt timer
    Allows to interrupt the advance of the timer.

    Reset timer
    Allows to reset the timer count.

    In the left section are instead present information regarding the machine and tools usage time.

    Total time

    Indicates the total work time of the machine.

    Disk time

    Show the disk utilization time.

    Fat

    To start the automatic greasing of the machine, use the following button:

    Start grease cycle
    Start a greasing cycle of 1 minute duration.
