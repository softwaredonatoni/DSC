3.11.2 CREATION OF RECTANGLES

    It is possible to build rectangular pieces by inserting the measures of width and height and to arrange them on the list of pieces to be worked.

    The program allows also to associate to each piece a mark (or label) identificative, and to set the number of repetition of rectangles with equal dimensions.

    Description of parameters

    For the creation of rectangular pieces the following data are required:

    Pieces Number
    Number of piece copies having the same measurements.

    Width
    Width of the rectangle. It is measured with the measurement unit reported to its right.

    Height
    Height of the rectangle. It is measured with the measuring unit reported to its right.

    Mark
    Label that is added to the piece to identify it. It’s an optional parameter, as it is not fundamental for the creation of the piece.

    Additional buttons

    The following side buttons also allow to carry out additional actions.

    Single importing
    If this button is enabled, it makes possible the creation of more distinct pieces in a single quantity instead of one piece with more copies.

    Dynamic Tag Top
    Assign to a side of the rectangle the Dynamic Tag Top.

    To activate the functionality relating to each button, it is necessary to press it. In this case, it will be highlighted with a light green color.

    Info

    NOTEThe 'Dynamic Tag Top' buttons that are located next to the width and height boxes are mutually exclusive. It is therefore possible to assign to only one side of the rectangle the Dynamic Tag Top.
