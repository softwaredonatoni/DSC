3.11.4 IMPORT EXCEL FILE (CSV)

    This section offers the possibility to the operator to import pieces starting from files in CSV format.

    Preview

    The list on the left of this section reports all the files in CSV format present in the currently selected path.
    The pressure of an element from the CSV files list will update the preview that compose the central part of this section.
    In the following example it is possible to notice the update of the preview following the pressure of the first element of the list:

    Proximity multipliers

    Is given the possibility to multiply by 10, or by 100, or by 1000, the size of the pieces present in the selected CSV file.

    The following image updates the example of the image reported in the chapter 'Preview', in which the values 'Width', 'height' and 'thickness' undergo a variation following the pressure of the second multiplier of protrusion button 'X 100':

    In this example it is also highlighted how the 'X 100' button after being clicked, is updated changing its background color with a light green hue.

    Sort

    The sorting functionality populates the sorting list based on the fields present in the selected CSV file and allows the operator, if desired, to apply a filter in alphabetical order or reverse alphabetical order based on one of the fields provided in the selected file.
