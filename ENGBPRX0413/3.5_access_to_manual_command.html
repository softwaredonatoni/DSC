3.5 ACCESS TO MANUAL COMMAND

    This panel allows access to some manual commands of the machine.

    Buttons

    0_ - 90_
    Rotate axis C to 0 or 90 degrees.

    Parking
    The machine moves to the parking position.

    Probe slab
    It will be possible to select any point on the slab, that will be the point where the probing will be executed.

    CRF cycle
    This button activates the procedure for the CRF calculation.

    Turning on / Turning off internal water
    Pressing this button activates the water from inside the tool.
    The color of the button indicates the state of the water: White = Off; Blue = On.

    Turning on / Turning off external water
    Pressing this button is activated the water of the main duct.
    The color of the button indicates the water status: White = Off; Blue = On.

    Slab probe cycle

    The keyboard cycle allows to calculate the thickness of the slab and take the bench quote using the suitable probe mounted on the machine.
    To start it, activate the button inside the manual commands panel, subsequently press a point inside the slab on the working area.
    In the chosen position, a symbol will be visible, which indicates the position in which the keyboard must be executed.

    Once the machine has finished executing the cycle, a panel will open that shows some measurements obtained through the slab probe.

    These information include:

    Thickness

    Slab thickness measured.

    Bench quote

    Bench work quote.

    Slab quote

    Measured quote on the top edge of the material placed on the bench.

    It is possible to save the data collected from the probe cycle using the buttons below:

    Set slab thickness
    Allows saving the slab thickness measured with the probe cycle.

    Set bench quote
    Allows to set as bench quote measured with the keyboard cycle.

    Start probing
    Restart the probe cycle in the same indicated position.

    CRF (Center Rotation Flange) measurement cycle

    The CRF cycle allows to measure the distance of the inner edge of the disk's tooth currently mounted on the machine, from the C axis of rotation.
    To start it, activate the button inside the manual commands panel, then press a point inside the slab on the work area.
    In the chosen position, a symbol will be displayed, which indicates the position where the cuts with disk must be performed to measure the CRF.

    Warning

    WARNING!During the execution of this cycle, cuts are performed in the material.Therefore it is recommended the operator to pay attention to the position of the slab in which to start it, to avoid damaging it.

    The machine will execute two parallel cuts with A axes to 0 degree: the first with C axes equal to 0 degree, while the second with C axes equal to 180 degree.
    At the end of the procedure, the operator is required to measure, using meter or caliper, the distance from the inside of the two cuts on the slab.
    This measurement must then be inserted inside the 'A' field of the panel that opens at the end of the cycle.

    The measures performed must be inserted in the appropriate field:

    A

    Internal distance between the two cuts performed during the cycle on the slab.

    CRF (A/2)

    Measure of the disk's Flange Rotation Center. It's calculated as half of the internal distance between the two cuts present on the slab.

    It is possible to save the data collected from the CRF measurement cycle using the following button:

    Set CRF Measure
    Allows to set as CRF measure of the disk currently mounted on the machine that calculated with the CRF cycle.
