3.13.19 MAXBOUND (OPTIONAL)

    This functionality allows to convert any piece in its square shape of maximum size, that is the smallest rectangle able to contain it entirely.
    The interface for this function looks like the following image:

    Parameters

    It is possible activate the following parameter clicking on the checkbox on its right.

    Find minimum

    This function allows to calculate the minimum footprint of the piece on the slab surface, optimizing its positioning.

    Execution of working

    Parameter deactivated

    Clicking on the 'OK' button, the piece will be converted into its rectangular version.

    Parameter activated
