2.6 CUTS

    With this functionality it is possible to add the multiple cut.
    This particular type of cuts allows to create a series of parallel cuts among themselves in the same group, with the possibility to have two groups each with its direction set.
    The interface appears similar to the following:

    Side bar

    Selection and management of cut groups

    Create first group of cuts
    Cuts are created parallel to the current position of the machine.
    Under the button is reported the direction in degrees for the first group of cut.

    Create second group of cuts
    The second group of cuts is created with a mode analogous to that of the first, with the difference that the direction may be different. Below the button is indicated the direction in degrees for the second group of cuts.

    Cut Management
    In this mode it is possible to apply some changes to the cuts, before they are performed. It is possible: modify, disable, move, delete and add pauses before performing the cut.

    Cutting parameters

    For each of the two groups of cuts available to the operator it is possible to insert up to 100 simultaneous cut configurations, with their respective parameters.

    Orthogonal directions
    If the two groups of cuts have orthogonal directions with each other, the first green icon will be displayed just below the selection and management buttons of groups of cuts. Otherwise, the second red image will be displayed.

    Width

    The distance between a cut and another.

    Numbers

    The quantity of parallel cuts to create.

    Location cuts
    These buttons are placed next to each cut configuration and allow the operator to decide whether to position the cut from the disk position in the motor direction, or vice versa to move away from the motor with the cuts.

    Show next/previous cuts
    Allows the operator to view all cuts of the selected group.
    Modify the selected group returns to the first page of the group, where the first four cut configurations of the group will then be visible.

    Stretching cuts

    The operator has the possibility to extend all cuts present in the first and second group at the beginning or at the end of the set measure.

    Start
    Acting on this parameter, you can extend the cut in its initial phase.

    End
    Acting on this parameter, you can extend the cut in its final phase.

    Delete and updates cuts

    Delete cuts
    Clicking this button the operator can delete the cuts created to restart with a new layout.

    Updates cuts
    At the pressure of this button the cuts will be moved in the correct position with respect to the machine.

    Axis C

    It is possible to rotate the head of the machine in order to adjust the orientation of the cuts of both the first and second group.

    Rotate _90
    The pressure of the button causes a rotation of 90_ of the machine head from the current position.

    Rotate -90
    The pressure of the button causes a rotation of - 90_ of the machine head from the actual position.

    Machine head rotation
    In the following parameter is possible to specify the angle (in degrees) of rotation of the machine's head.

    MDI
    The pressure of this button activates the rotation of the machine head according to the value set in the previous parameter.

    Thickness

    The operator can go to define thickness of the slab that will be cut.

    Manual thickness
    The operator can insert manually the thickness of the slab.

    Thickness with machine
    The operator can acquire the thickness using the machine bringing the height of the machine to the desired quote and selecting the button. The distance between the bench and the disk will be considered as material thickness.

    Execution of working

    The operator has the possibility to set both groups of cuts modifying their relative parameters.
    Once the desired result is obtained, the 'Start cycle' button can be pressed to execute the working.

    First group cut

    In the first group of cuts will be defined one or more groups of cuts modifying the parameters Width, Number and Position Cuts, also the rotation of the C axis.
    In the following working there have been set parallel cut with respect to the current position of the machine.

    Second group cuts

    Similarly to the first group of cuts, also for the second it is possible to change the parameters to obtain the researched result.
    In this working the direction of the cuts was modified, executing some cuts orthogonal to those of the first group.

    Optimization and cut management

    In this section we can go to modify the execution order of the cuts with the possibility of choosing whether or not to execute a cut.
