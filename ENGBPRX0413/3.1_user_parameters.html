3.1 USER PARAMETERS

    Once the button is pressed, it is possible to view various windows.

    Windows

    There are 4 different windows that will be individually illustrated later.

    User Values

    Window dedicated to changing some settings related to workings.

    Statistic

    The operator can view the statistics of the workings performed.

    Layers name

    Window dedicated to the settings for the import of DXF files on Parametrix.

    Suction cups

    Parameters relative to the vacuum movement speed.

    Bottom bar

    Close Parametrix
    Closes the Parametrix software.

    Restart Parametrix
    Restart the Parametrix software.

    Probing
    Opens the Windows keyboard.

    Camera connection
    Opens the camera management software.

    Remote Assistance
    Open the connection for the remote assistance.

    Technical Parameters
    Opens the technical parameters page. Intended for use by a technician.

    Password insertion
    Allows to insert the password.

    Don’t save changes
    It exits from the page without saving changes.
