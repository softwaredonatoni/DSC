3.13.12 SURFACE WORKINGS (OPTIONAL)

    This feature allows the operator to perform workings on the surface of the material.

    Working types

    To use this working, it is necessary to first choose one of the two available modes, selecting the corresponding button:

    Info

    NOTEThe ""Rectangular surface"" and ""Zigzag surface"" modes are mutually exclusive: activation of one automatically excludes the other.

    Rectangular surface
    The machine will execute parallel passes to move along the surface.
    For this working is used the disk.

    ZigZag Surface
    The machine will execute zig-zag passes to move along the surface.
    For this working is utilized the mill.

    Parameters

    Axis C Rotation

    Determines the orientation of the working which can be horizontal (0_) or vertical (90_) with respect to the work plane.

    Info

    NOTEThe modes '0_' and '90_' are mutually exclusive: activation of one automatically excludes the other.

    Pass Extension

    Each cut generated from this working is prolonged, both in head and in tail, of the length indicated by this parameter.

    Depth cutting

    Indicates the depth of the working in the material.

    Execution working

    Info

    NOTEIt is possible to apply the processing by pressing the button on the left of the suitable cut in the list of polygons.An icon red with a 'X' indicates that the process can’t be applied to the desired cut.

    Rectangular Surface Execution

    After having set the processing parameters, select a side of the piece on which you want to execute the processing.
    The cuts will be made from bottom to top.

    Surface ZigZag Execution

    After having set the processing parameters, select a side of the piece on which you want to execute the processing.
    The cuts will be made from bottom to top.


    Slab processing parameters

    The operator can modify the parameters linked to the superficial workings from the parameters panel of the slab processing.

    Rectangular surface

    This screen shows the processing parameters for the 'Rectangular surface' mode.

    Description parameters

    Distance between steps going
    Represents the distance between the outgoing pass and the next returning pass.

    Distance between returns
    Represents the distance between the return pass and the next return pass.

    Only forward steps
    If enabled, the passes will be carried out all in the going direction, that is from the left to the right of the piece.

    Only Steps back
    If enabled, the passes will be executed all in the return direction, that is from the right to the left of the piece.

    Raises disk at the end of the pass.
    If enabled, at the end of each pass the disk is lifted.

    Piston Pressure
    Allows to enable pressure of the piston on the material, and decide its intensity

    Info

    NOTEThe enabling of parameters for working passes in a specific direction are mutually exclusive.If one of the two is activated, the program forces also the activation of the disk lifting at the end of the pass.

    Description parameters speed

    Penetration
    Determines the speed during the phase of penetration of the tool on the material.

    Forward
    Indicate desired speed for the cutting phase in approach.

    Back
    Indicate wished speed for the cutting phase in the return.

    Zigzag Surface

    This screen shows the processing parameters for the 'Zigzag Surface' mode.

    Description parameters

    Distance between steps going
    Represents the vertical distance between the starting point of the past diagonal and its arrival point in the going direction.

    Distance between returns
    Represents the vertical distance between the starting point of the previous diagonal and its arrival point in the return direction.
