    Selection working type

    The selection of the working type adjusts the content of the parameters management window.

    For each working type is visible in its relative preview the selected increment mode, the cutting speed forward (of plunge / entrance in the case of working type Hole) and the diameter of the tool.

    Type steps

    Full Step
    The Full Pass provides the execution of all working in a single pass. Consequently, the tool immediately enters the entire material and executes the working.

    Step / spiral
    These passages foresee to not enter up to the end of the material right away, but gradually.
    This means the execution of the processing several times at different heights, until reaching the workbench quote (end of material) or the decided height.

    Type of workings

    Linear

    Full Step

    Step to step

    Description parameters increments

    Forward
    Indicate the depth of cutting on the material when the machine cuts towards the sense of rotation of the disk.

    Back
    Indicates the depth of the cut on the material in the cutting phase in the direction opposite to the direction of the tool.

    Last
    Indicate the cutting depth that the tool’s last pass must have on the material. That cut is always in the opposite direction compared to the movement of the disk.

    Parameters description speed

    Penetration
    Determine the speed during the penetration phase of the tool on the material.
    Such measurement is expressed in millimeters per minute.

    Forward
    Indicate desired speed for the cutting phase in approach.
    Such measure is expressed in millimeters per minute.

    Back
    Indicate desired speed for the cutting phase in the return.
    Such measurement is expressed in millimeters per minute.

    Last
    Indicate the speed that the machine must keep on the cut in the last pass.
    Such measurement is expressed in millimeters per minute.

    Accelerate cuts
    If this parameter is set with a value greater than zero, the rotation speed of the disk will be progressively increased, starting from the beginning of the cut up to the desired distance.
    Subsequently, the rest of the cut will occur at full regime.

    Slow down cuts
    If this parameter is set with a value higher than zero, the disk rotation speed will be progressively decreased starting from the distance set that separates from the end of the cut, until the end of the cut.

    Info

    NOTEAs for increments and speeds the parameters 'Backward' and 'Last', can be present or not or only one of the two can be present.Normally these choices are made at the program installation moment and are decided based on the client's working.It is possible to change these options, as you prefer to work differently by contacting Donatoni Assistance.

    Execution working with step pass:

    • The machine will approach the material at maximum speed, until the achievement of the distance from the material specified in the parameter APPROACH.

    • The number of steps will be calculated as:

      • Thickness material divided thickness of steps

        • eventually rounded up

    • The machine will begin to move at working speed.

      • If the STEP INGRESSO parameter value is set (different and greater than zero), the machine will work at the speed set in the second INGRESSO field until the distance set in the STEP INGRESSO field is reached.

    • The machine will execute the steps calculated at the second point at Step speed

    • After each step (included the first one) the machine will return to material quote at maximum speed, then it will return to maximum speed at short distance compared to the beginning of the next step.

    • If the STEP OUT parameter is set (different and greater than zero), the machine will work at the speed defined in the second STEP field until the distance defined in the first OUT field is reached.

      • The rest of the Step will be executed at EXIT speed

    Description parameters Hole with step pass

    Approach quote
    The machine will approach the material at maximum speed until it reaches the distance from the material, set in this parameter. From this moment the drill will start to move at processing speed.

    Step
    These parameters define the thickness of the work steps and the work speed. Furthermore, the number of work steps is calculated based on the thickness of the work steps and the material thickness.
    For example if for thickness of 20 mm of a material the user sets the parameter to 3 mm, the hole will be executed in 7 steps.

    Way in
    If this value is set different than 0, an interval will be defined starting from the first step and eventually for the next steps, in which the machine will work at a speed different from the one defined in 'STEP'

    Exit
    If this value is set different from 0 a range will be defined starting from the last step and eventually for the previous steps, in which the machine will work at a speed different from the one defined in 'STEP'

    Pause after
    If this parameter is activated, the machine will perform a pause after each execution of a hole to allow the operator to check any material residue remaining blocked inside the tool.
    The operator will press again the button START to allow the continuation of the program.

    Change of cutting parameters

    It is possible modify the previously illustrated parameters for the disk, the milling cutter and the drill, if the user is enabled to use them.

    To decide which tool to modify the parameters, it is necessary to select one through the pressure of the button visible in the right part of the 'Select tool' panel.

    Change cutting parameters
    Allows access to the screen of change of the cutting parameters of the selected tool.

    Pressing this button a window will be visible that will contain a configuration of saved parameters for each material present. The results depend also from the selected working type.
