2.8 POLISH (OPTIONAL)

    The Polishing program allows to insert the polishing with different working types. It is possible, moreover, to choose the geometry acquisition method of the slab: through the detection of the points that determine the vertices of the surface (from three up to a maximum of six) or using a regular surface (rectangle) through the entry of the zero quota and the value of the two sides.

    Processing mode

    To use this working, it is necessary to choose one of the two available modes, selecting the corresponding button:

    Info

    NOTEThe modes 'Rectangle Selection' and 'Free point selection' are mutually exclusive: the activation of one automatically excludes the other.

    Switch to free point selection mode
    Pressing the button in figure it is possible to switch from the working mode in which the rectangle is used as the predefined figure, to that in which the surface acquisition happens through the selection of the points.

    Switch to rectangular selection mode
    Pressing the button in the figure allows to switch from the working mode where the surface acquisition happens by means of point selection, to the one that uses the rectangle as a default figure.

    Way in/Way out

    The buttons in figure allow to change the entrance and the exit of the tool between progressive and standard mode.

    Info

    NOTEThe modes 'Progressiva' and 'Standard' are mutually exclusive: enabling one automatically excludes the other.

    Pass to progressive mode
    Enabling this mode, at the start of the program, the machine will move from the work Z height to the safety Z height, then towards the first point of the surface to be polished (for the rectangle it is the one in the upper left). Subsequently it will move along the perimeter of the surface, gradually lowering the Z height so that, at the end of the path, the tool is at work height. After this phase, the machine will perform a polishing machining on the perimeter, after which the selected machineries will be executed in sequence. At the end of the working sequences, the machine will make a pass on the perimeter of the surface to be polished, then performing the same path rising gradually in Z until reaching (at the end of the path) the safety height.

    Switch to standard mode
    Enabling this mode, at the start of the program, the machine will move from the working Z height to the safety Z height, then heading towards the first point of the surface to be polished (for the rectangle it is the one in the upper left). Subsequently, keeping the X and Y axes fixed, it will lower in Z to the working height, then the selected polishing operations will begin. At the end of the set working sequences, the machine will move towards the first point of the surface to be polished, then make the ascent in Z to the safety height.

    Free points selection mode

    This mode is displayed as follows:

    Parameters

    Points location

    Set (Imposta)
    This button allows the insertion of coordinates of each point. When the position has been acquired, the status of the figure on the side changes from red to green.

    Info

    NOTEThe input of points goes from a minimum of three to a maximum of six.

    Clean
    This button allows to delete all the quotes saved if the setting is not correct and then you want to acquire again the points.

    Working types

    Perimeter
    The polishing operation will affect exclusively the perimeter area of the geometric figure set.

    Horizontal passes
    Multiple horizontal passes will be executed.

    Vertical Steps
    Multiple vertical passes will be executed.

    Global Repetition
    It is possible to restart the processing cycles by entering the number of repetitions in the appropriate box. In the box relating to the 'Performed' column, the number of repetitions performed of the complete processing is instead shown.

    Machining parameters

    Steps

    Each working type can be repeated a number of times set in this dedicated field.
    If the value is set to '0', the processing in question is not performed.

    Made

    The number of steps performed is displayed.

    Step [mm]

    Indicates the distance between one pass and the next, selectable for each working type.

    Parameters tool for polishing

    Work speed
    Indicate the speed with which the working will be executed.

    Drop speed
    Indicate the desired speed in the positioning along the Z axis at the selected coordinate.

    Tool wear
    Allows to take into account the tool wear in working phase, inserting its compensation value.

    Execution mode Free point selection

    Warning

    WARNING!Between a working and the next one the machine executes the movements without rising to the safety height.

    The operator sets the position of a minimum of three up to a maximum of six available points, moving the machine manually and using the relative saving commands. Completed the definition of the points, you proceed with the configuration of the machining parameters for each desired polishing type. Contemporarily, also the specific parameters of the tool to be used for polishing are set.

    At any time before the start of the polishing cycle, the operator has the possibility to change the tool entry and exit mode by selecting one of the two available options.

    Once configuration is complete, it is necessary to tilt the A axis of the machine to 90_ and start the rotation of the disk. The operator can then proceed with the execution of the programmed cycles by pressing the 'Start cycle' button. During execution, the progress status can be monitored through the 'Completed' column, which reports the completed cycles for each selected polishing type.

    Rectangular selection mode

    This mode is shown as follows:

    Parameters

    Rectangle position

    Sides of rectangle
    The figure on which the working is carried out is determined by the values attributed to Y and X which correspond to the sides of the rectangle. The coordinates of the lower left vertex of the rectangle are acquired through the respective reset buttons, present below each axis. It is necessary that both values are reset.

    Machining parameters

    It is postponed to processing parameters Free point selection for the workings: Contour, Horizontal/vertical passes and Global repetition.

    A zig zag
    Zig Zag passes will be executed.

    Parameters of the tool for polishing

    It is referred to Disk Parameters Free point selection for the disk parameters: Working speed, Descent speed and Disk wear.

    Y axis step for ZigZag
    For the zigzag polishing mode, it is possible to modify the processing step in Y, allowing to break the work trajectory into more passes. Keeping the value at zero or setting a value greater than that attributed to parameter B, the polishing step along the Y axis will coincide with the value of B.

    Execution mode Selection rectangular

    Warning

    WARNING!Between a working and the next one the machine executes the movements without rising to the safety height.

    The operator defines the position of the rectangle intended for polishing, moving the machine manually and modifying the values of the corresponding coordinates. Once the rectangle definition is complete, proceed with the configuration of the machining parameters for each type of desired polishing. At the same time, the specific parameters of the tool used for polishing are also set.

    At any time before the start of the polishing cycle, the operator has the possibility to change the tool entry and exit mode by selecting one of the two available options.

    After configuration is complete, it is necessary to tilt the A axis of the machine to 90_ and start the rotation of the disk. The operator can then proceed to execute the programmed cycles by pressing the 'Start cycle' button. During execution, the progress status can be monitored through the 'Completed' column, which reports the completed cycles for each type of polishing selected.
