3 PARAMETRIX

    The system has been designed to facilitate the operator in the programming of the automatic cuts. Through the software is possible:

    • Determine the position of the slab in the machine's work area and create the material perimeter for the processing.

    • Create pieces through parametric figure

    • Import figures realized with other CAD

    • Generate a list of pieces to work

    • Search the best position of the pieces on the slab

    • Set special cutting functions

    • Program the cutting order and create the machine handling

    • Monitor video the execution of the cuts

    Work area and piece list

    Below is reported a typical representation of the Parametrix working area with the list of imported pieces already populated and an image of the acquired slab.

    List pieces inserted

    When a piece is created the program adds it on the piece list; from the list it is possible to insert them on the working area or to make some changes to the piece and to check the number of missing pieces to the process.

    Pieces positioning

    Parametrix allows to position pieces inside the work area of the machine to create the cut file.
    Once acquired the slab, created the perimeter and created the pieces of the working, it is necessary to import the pieces inside the slab, with the button on the pieces table the piece is put inside the working area. Is possible work with 1 or more pieces contemporaneously.

    In the case in which the pieces are not perfectly inside the slab, a warning/danger signal will appear in the top left corner.
    The appearance of this signal indicates that problems may arise in the phase of management of the cuts, however no function of the program will be blocked.

    Top bar

    In the upper part of the screen, in the center and on the left are visible the following buttons, preceded by the logo and the time marked:

    The upper bar of commands is integrated with some buttons that will be illustrated later:

    User parameters
    Opens the page of the technical parameters. Designed for the use of a technician.

    New working
    The pressure of the assigned button to conclude a working and start the next one calls a confirmation pop-up.

    Parameters tool
    Allows the operator to manage the tools.

    Slab processing parameters panel
    The pressure of this button allows the passage from the ‘Tool Panel’ to the ‘Slab Panel’.

    Access to manual command
    You can access a panel with manual controls like making the axis rotate and parking the machine.

    Laser Management
    This function activates the slab acquisition menu.

    Add photo to bench
    With this function you can take a photo from the slab or choose it from the files to place it on the work area.

    Slab perimeter
    This function allows to trace the perimeter.

    Bench viewing management
    This function opens a panel with tools to resize the working area.

    Viewing of work entities
    This function opens a small panel with viewing tools.


    Side section

    The buttons located in the top right corner allow the operator to change the Parametrix screen.

    Each screen is responsible for a particular aspect of the processing of a slab.

    Placement of pieces
    Main screen of Parametrix.
    Operations of:

    • creation and modification of the pieces

    • acquisition of the slab photo on the bench and its perimeter;

    • placement of pieces on the working area

    Piece cutting order management
    Screen for managing the order of cutting the pieces.

    Optimization and cut management
    Screen for the optimization and cut management.
    Operations of:

    • sorting and editing of the cuts

    • handling of the material with vacuum

    • bench washing

    • start of the program of cut of the machine;

    On the bottom right are the following buttons whose functionalities will be seen later.

    Edit piece
    Panel through which it is possible to apply additional workings to pieces.

    Activate/deactivate standard pieces
    Allows to hide or view the rows of the list associated with commercial pieces.

    Piece creation
    Panel that manages the creation and the import of the pieces on Parametrix.

    Order or filters pieces
    Function that allows the operator to apply a filter and/or order the pieces of the list.

    Delete list of pieces
    Allows to delete all content of the pieces list.

    Delete piece
    Allows to delete the pieces from the piece list.

    Modify additional piece properties
    The operator can assign and change the additional properties of the selected piece.

    Lower bar

    Upon opening the 'Parametrix' functionality, the bottom bar will undergo some changes as shown below:

    This bar is integrated with these buttons:

    Multiple selection
    With this function it is possible to select or deselect more pieces at once.

    Select all
    With this function you can select or unselect all the pieces present on the working area.

    Remove piece
    Allows to remove the pieces from the processing area.

    Saving
    Allows you to save the status of what you are working on.

    Load
    Allows to load a project or a working.

    Information Panel
    A panel opens with various information.

    Restore modify
    Cancel the last operation carried out.

    Machine on cut
    Allows to control the position of the piece in the work area with respect to the position in machine.

    Piece rotation
    Panel with which is possible rotate the piece or the selected pieces of 90 degrees or of tot degrees at a time in clockwise or counterclockwise.

    Move pieces
    Panel with which is possible move in the various directions the piece.

    Magnetize pieces
    Functionality that allows to attach two cuts to each other.

    Miter cut
    Panel that allows to make a miter cut on a piece in the working area.

    Edit standard piece
    Panel with which you can change the dimensions of a commercial piece.
