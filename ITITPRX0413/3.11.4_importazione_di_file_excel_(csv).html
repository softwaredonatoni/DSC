3.11.4 IMPORTAZIONE DI FILE EXCEL (CSV)

    Questa sezione offre la possibilità all’operatore di importare pezzi a partire da file in formato CSV.

    Anteprima

    La lista sulla sinistra di questa sezione riporta tutti i file in formato CSV presenti nel percorso attualmente selezionato.
    La pressione di un elemento dalla lista dei file CSV aggiornerà l’anteprima che compone la parte centrale di questa sezione.

    Ordinamento

    La funzionalità di ordinamento popola la lista di ordinamento in base ai campi presenti nel file CSV selezionato e permette all’operatore, se lo desidera, di applicare un filtro in ordine alfabetico oppure alfabetico inverso basato su uno dei campi previsti nel file selezionato.
