3.1.3.3 DYNAMIC TAGS

    Con questa funzione è possibile assegnare dei parametri di configurazione ai Layer dei file DXF.

    Parametri

    I parametri sono:

    Name

    Serve per definire e riconoscere il nome del Layer.

    Entity Type

    Definisce la tipologia delle entità che verranno visualizzate.

    LineWeightMethod

    Permette di scegliere la modalità con cui viene ricavato lo spessore delle linee.

    LineWeight

    Permette di definire lo spessore delle linee appartenenti al Layer.
    Questo spessore viene applicato se LineWeightMethod è impostato in modalità “ByConfiguration“.
    Parametrix utilizza di default uno spessore di 2mm per le linee.

    ColorMethod

    Permette di scegliere la modalità con cui viene ricavato il colore del Layer.

    Color

    Permette di definire il colore delle entità appartenenti al Layer.
    Questo colore viene applicato se ColorMethod è impostato in modalità “ByConfiguration“.

    Pulsanti

    Con i seguenti pulsanti è possibile aggiungere o eliminare una configurazione:

    Crea configurazione
    Permette di creare una configurazione di parametri per l’importazione di un Layer.

    Elimina configurazione
    Permette di eliminare una configurazione di parametri per l’importazione di un Layer.
