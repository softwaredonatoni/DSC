2 DSLABS

    The system allows the user to configure, load and view slab information. Through the software it is possible:

    • Load new single slabs

    • Load multiple new slabs with the same configurations

    • Manage loaded slabs

    • Set additional features

    Side bar

    This bar appears in all main pages present in it: 'Home', 'Browse Slabs' and 'Settings'.

    Menu
    This button allows alternating the expansion and contraction of the side bar.
    When the sidebar is wide the button names will be displayed.

    Home
    Allows returning to the Home screen viewing.

    Browse Slabs
    Pressing this button will open the 'Browse Slabs' screen, as for the dedicated card in the Home Screen.

    Machine settings
    This button allows access to the system parameters page.

    Home Screen

    At startup of the application the following initial screen is displayed:

    The cards listed below allow performing various actions by clicking on each of them.
    Each card will be individually analyzed in the following chapters.

    Take photo
    Selecting this card will open a window to load a slab photo via a connected camera.

    Open Photo
    When this card is selected, the user can select an image from the device memory.
    The selected image is imported as slab; subsequently a new window is opened that allows to set its parameters.

    Browse Slabs
    Selecting this card opens the 'Slabs List' page, allowing all slabs added to the system to be viewed.
    From this page the user can search for slabs and view them, modify them, export them or delete them.

    Open Photo

    This function allows the user to import the image of a slab by selecting it directly from the device memory.
