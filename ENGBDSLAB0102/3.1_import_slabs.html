3.1 IMPORT SLABS

    In the following section the 'Massive Slabs Import' optional is described, such optional allows loading multiple identical slabs into the system. where the system will handle the enumeration of the Slabs Names. This optional may be requested to be added during the first installation of our operator.

    Enabling this optional will display the 'Homepage' screen as follows:

    Card

    A new selectable card has appeared:

    IMPORT SLABS
    The following card will open a new window similar to that of 2.3 Slab Edit with the addition of some additional parameters

    Window

    At the opening of the window the screen will be the following:

    As you can see the screen is similar to that of Slab Edit, but with some differences

    Preview slab

    Unlike other slab addition systems to the system (Take photo and Open photo), it is possible to load the slab without an image by creating a slab simulation with just a rectangle indicating the slab dimensions.

    The size of this image is managed by the parameters in 'Bulk Slab Import' (currently closed dropdown menu), furthermore other image data can be configured, such as background color, slab color and the distance of the slab from the bottom edge.

    Regarding the slab perimeter it is modifiable through the basic parameters of 'Width' and 'Length'.

    Slab preview becomes Edit slab if we import a slab image through the 'Open photo' button then the difference between 'Edit slab' and 'Import slabs' will only be the ability to mass import slabs of the same type.

    This is the situation if you had an image slab to massively import:

    Parameters

    Most of the parameters in both cases work exactly like the parameters in the chapter of Slab editing, the data we are going to describe are those in the subsection 'BULK SLABS IMPORT'

    Base parameters mass import

    The parameters of this group are those that appear both when creating a 'virtual' slab and when importing an image of a slab

    NOTE

    The symbol '*' next to the names indicates that the parameter is mandatory

    Progressive code separator
    In the slab name insert the separator that divides the **Slab name** parameter and its enumerative

    From
    Set the starting number for slab enumeration

    N_ copies of the slab
    Set how many copies to perform

    Warning!

    Insert a number of copies less than 1000

    Parameters slab 'VIRTUAL'

    The parameters of this group appear if you are performing a bulk import without an image:

    NOTE

    The symbol '*' next to the names indicates that the parameter is mandatory

    Image width
    Set the width of the entire image

    Warning!

    This value must be greater than the slab width

    Image height
    Set the width of the entire image

    Warning!

    This value must be greater than the slab height

    Background color
    Set the background color of the 'virtual' image

    Slab color
    Set the background color of the 'virtual' slab

    Distance bottom edge
    Set the distance of the 'virtual' slab from the bottom edge of the image
