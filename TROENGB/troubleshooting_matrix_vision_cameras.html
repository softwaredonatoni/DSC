TROUBLESHOOTING MATRIX VISION CAMERAS

    ATTENTION: Only attempt the following AFTER confirming that the issues are indeed related to the camera and not to DTwain.

    Basic Steps:

    1. First and foremost, for any issue, ensure the camera is properly connected without connection problems. To do this, open the mvIPConfigure program.

    2. The camera should be displayed in the device list; otherwise, it's not connected to power or the router.

    3. If you click on the camera, it should indicate that there are no connection problems. If problems are detected, an automatic resolution procedure is usually suggested.

    4. A series of parameters are displayed on the right, including the assigned IP and the negotiated connection speed. The latter MUST be 1000 mb (or 1 gb). If it negotiates less (e.g., 100mb), there's very likely a problem with one of the two Ethernet cables (camera ↔︎ router or PC ↔︎ router). Test with a spare cable to see if replacing one of them can solve the problem.

    Output photo with strange lines and/or colors

    Symptom: The photo is taken but with strange lines and unusual colors (usually green/purple lines).

    Solution: AFTER trying the basic steps above, you can try updating the camera's firmware.

    mvBlueCOUGAR-X_Update-250330.mvu

    1. Download the file above.

    2. Copy it to the client's desktop.

    3. Open the mvDeviceConfigure software.

    4. Select the camera.

    5. Select Action -> Update Firmware -> From Specific Local Firmware Package.

    6. Select the file from the desktop.

    7. Confirm.

    8. Wait for the procedure to complete (it takes a few minutes).

    9. Verify if the problem is resolved.
