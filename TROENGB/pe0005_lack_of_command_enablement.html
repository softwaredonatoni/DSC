PE0005 LACK OF COMMAND ENABLEMENT

    • Explanation of the error
    • Possible causes
    • Possible solutions
    • Related articles

    Explanation of the error

    It is displayed when the machine is turned on or after the emergency button is released.

    Possible causes

    1. The command enable button (RESET) has not been pressed;

    2. Malfunction of the command enabling button (RESET) on the electrical panel;

    3. Pilz safety control unit malfunction.

    Possible solutions

    1. Press the reset button on the electrical panel

    2. Check the contacts of the command enabling button, i.e. when I hold down the button the contact must close and the green start LED must appear on the control unit.
      If necessary, replace the button contacts;
