DDX _ STATISTICHE PARAMETRIX

    Nei file iso che genera DDX è possibile avere una parte iniziale che riporta informazioni sulle statistiche

    Quando il file iso viene caricato dal Parametrix questa parte viene interpretata e si generano delle statistiche più complete

    Iso

    SENZA NESTING

    %
; Postproc version: 1.0.4
;STATISTICHE
;LunghezzaRapidi=506
;LunghezzaLineari=575.477
;PreforiWJ=2
;Fori=3
INCLUDE "vars_inc.cfs"
RTCP=1
G600 X0.000 Y0.000 Z-30.000 U500.000 V300.000 W0.000 O19
G601
G64
SETMSP=1]]>

    Mapping

    l’interpretazione delle statistiche ddx è stata portata su due file di mapping che si trovano al path:

    C:\Parametrix\CSV\Mapping

    Per le statistiche

    DDXMapStatistiche.csv

    ddx;statistiche;scala
LunghezzaRapidi;LunghezzaRapidi;1
LunghezzaLineari;LunghezzaLineari;1
PreforiWJ;PreforiWJ;1
Fori;Fori;1
Spessore;Spessore;1
AreaLastra;AreaLastra;1000000
VolumeLastra;VolumeLastra;1000000000
AreaPezzi;AreaPezziCommesse;1000000
Resa;ResaCommesse;1
MetriLineariTaglio;MetriLineariTaglio;1000
MillimetriQAreaTaglio;AreaTaglio;1000000
MetriQAreaTaglio;AreaTaglio;1000000
DurataStimata;DurataLastra;60
NomeLavorazione;NomeLavorazione;1
DataInizioLastra;DataInizioLastra;1
DataGenerazionePGM;DataGenerazionePGM;1
NomeLastra;NomeLastra
LarghezzaLastra;LarghezzaLastra;1
AltezzaLastra;AltezzaLastra;1
NumeroPezzi;NumeroPezzi;1]]>

    per le statistiche pezzo

    DDXMapStatistichePezzo.csv

    ddx;statistiche;scala
AreaPezzo;AreaPezzo;1000000
VolumePezzo;VolumePezzo;1000000000
Nome;Nome;1
Altezza;Altezza;1
Larghezza;Larghezza;1]]>

    Il ribaltamento avviene secondo questi file segue la logica di mappatura tra il tag ddx nel file iso e la proprietà nelle statistiche con l’eventuale scala da applicare

    es:

    AreaPezzi;AreaPezziCommesse;1000000

    Tag in iso= ;AreaPezzi=2160000.0

    Map applicato: Statistiche.AreaPezziCommesse = AreaPezzi/1000000

    NB: la scala è un divisore

    Es dei file csv di mapping

    DDXMapStatistiche.csv

    DDXMapStatistichePezzo.csv

    CSV di Visualizzazione

    Al path: C:\Parametrix\Statistiche\VisualizzaStatistiche\Mapping

    Possiamo trovare i due file che pilotano la visualizzazione delle statistiche in modalità semplificata o full:

    Semplificata: MapColumnStatisticsSimplified.txt

    MapColumnStatisticsSimplified.txt

    Full: MapColumnStatistics.txt

    MapColumnStatistics.txt
